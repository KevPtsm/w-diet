Comparison: Weight Input Approaches

Option A: Tap Weight Card

  • PRO: Cleanest UI (no extra buttons)
  • PRO: Follows iOS patterns (cards are tappable)
  • PRO: Discoverable (hint text guides user)
  • CON: Might not be obvious it's tappable at first
  • CON: Accidental taps possible

Option B: Edit Button on Card

  • PRO: Explicit affordance (pencil icon = editable)
  • PRO: Prevents accidental input triggers
  • PRO: Consistent with eating window edit pattern
  • CON: Extra visual element (less minimal)
  • CON: Small tap target (icon only)

Option C: Notification-Driven

  • PRO: Proactive guidance (app tells user what to do)
  • PRO: Builds daily habit (morning routine)
  • PRO: Prominent CTA (can't miss it)
  • CON: Takes up dashboard space
  • CON: Requires notification permissions
  • CON: Feels pushy if user doesn't want to log weight

Best UX: Combine Option A (tap card) + Option C (notification prompt on Day 1 only).
